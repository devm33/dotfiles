# Copilot Instructions

## PR Comment Etiquette

- Never leave top-level comments on PRs. Only reply within existing review comment threads.
- When replying with a fix commit, reply with only the short SHA — no backticks, no formatting — so GitHub auto-links it.
- Always add _Generated by Copilot_ to the end of any comment posted

## PR Creation Conventions

- Always use the repo's pull request template when creating PRs.
- Follow repo conventions for commit messages, PR titles, and PR descriptions.
